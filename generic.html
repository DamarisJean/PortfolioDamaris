Ski-Safe PHP Laravel React MySQL


Ski-Safe is the full-stack project that I had the pleasure of creating alongside my team during a full-stack bootcamp. We implemented new technologies and worked together with clear communication and teamwork.
Ski-Safe is a 3D map that allows you to add a pin and explain the level of severity at those coordinates.
It also provides the real temperature of the area.
Only registered and logged-in users can create these pins, and admins can delete and edit them.
